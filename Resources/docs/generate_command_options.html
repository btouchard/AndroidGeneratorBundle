Command options

Build options :

  • -t | --target_sdk: Android target SDK
    Default: 24
  • -m | --min_sdk: Android min version SDK
    Default: 21
  • -s | --sdk_path: Android SDK path
    Default: /opt/android-sdk
  • -g | --gradle_version: Gradle version used to create project
    Default: 2.2.0

Android options :

  • -a | --app_name: Android application name
    Default: based on Bundle name
  • -p | --app_path: Path where store the Android application
    Default: in "android/{app_name}" on root project directory
  • -d | --app_domain: Android application target domain name
    Used for API url and package name if empty
  • -k | --app_package: Android application package name<
    Default: construct from domain name and app name/li>
  • -u | --api_url: Android application RESTfull API url
    Default: construct from domain name and app name
