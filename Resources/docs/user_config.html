Step 5) FOSUser configuration

Now, you must be create the User Entity class defined in FOSUser configuration part.
Example: AppBundle/Entity/User.php

<?php
namespace AppBundle\Entity;

use Doctrine\ORM\Mapping as ORM;
use FOS\UserBundle\Model\User as BaseUser;
use JMS\Serializer\Annotation as Serializer;

/**
 * User
 * @ORM\Entity
 * @ORM\Table(name="user")
 */
class User extends BaseUser {

    const ROLE_SUPER_ADMIN = 'ROLE_SUPER_ADMIN';
    const ROLE_ADMIN = 'ROLE_ADMIN';
    const ROLE_USER = 'ROLE_USER';

    /**
     * @var integer
     * @ORM\Id
     * @ORM\Column(name="id", type="integer")
     * @ORM\GeneratedValue(strategy="AUTO")
     * @Serializer\Expose()
     */
    protected $id;

    /**
     * @var string
     * @ORM\Column(name="firstname", type="string", length=255, nullable=true)
     * @Serializer\Expose()
     * @Serializer\Groups({"Me"})
     */
    protected $firstname;

    /**
     * @var string
     * @ORM\Column(name="lastname", type="string", length=255, nullable=true)
     * @Serializer\Expose()
     * @Serializer\Groups({"Me"})
     */
    protected $lastname;

    /**
     * @var string
     * @ORM\Column(name="city", type="string", length=255, nullable=true)
     * @Serializer\Expose()
     */
    protected $city;

    /**
     * @var string
     * @ORM\Column(name="country", type="string", length=255, nullable=true)
     * @Serializer\Expose()
     */
    protected $country;
}

Next create serialization model for JMSSerializer
Create file Resources/serialize/Model.User.yml, and put this for exposing data from FOSUserModel base class

FOS\UserBundle\Model\User:
    exclusion_policy: ALL
    properties:
        username:
            expose: true
        salt:
            expose: true
            groups: [Details]
        email:
            expose: true
            groups: [Me]
        enabled:
            expose: true

Now, update database:

php bin/console doctrine:generate:entities AppBundle
php bin/console doctrine:schema:update --force
php bin/console cache:clear

Step 6) Test your api

Create a new user in application:

php bin/console fos:user:create ben

And check login to API:

curl -X POST http://api.url/api/login  -d username=ben -d password=ben

You must be see a response containing JSON token like this:

{"token":"eyJhbGciOiJSUzI1NiJ9..."}
