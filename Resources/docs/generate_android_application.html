Step 9) Generate Android application from your Bundle

YES, you have done.

Now you are ready to generate an new Android application from your bundle application.

AndroidGeneratorBundle provide a console command to generate application.
php bin/console generate:android:app

You juste need to pass Bundle name as command argument, like this:

php bin/console generate:android:app AppBundle

See other options here

The script start and prompt somme information like

  • App name: Android application name
  • App path: Where is stored application
  • App domain name: DNS associate to the application (used two for package name)
  • App API url: URL of previously created API
